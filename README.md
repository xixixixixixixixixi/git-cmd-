#### 一.常用命令
1. 创建空文件夹
> $ mkdir 文件夹名称   例：mkdir sun 
2. 显示当前路径
> $ pwd         
3. 显示当前目录
> $ ls                
4. 显示隐藏文件
> $ ls -a      
5. 提示下一步操作     
> $ git status   
6. 回到根路径
> $ cd ~
